Maya用カスタムグラフエディターです。
ダウンロードは右上の緑色のCodeのDownload zipからできると思います。
フォルダ名をcustomGraphEditor-masterからcustomGraphEditorに変更します。
customGraphEditorのフォルダごとドキュメン > maya > scriptの中に入れ、Mayaを再起動後

from customGraphEditor import UI
UI.main()

とpythonで入力すれば起動できます。
エラー・改良してほしいところなどありましたら、issuesに書き込んでいただくか、twitterの@3D_manuあてにメッセージお願いします。
